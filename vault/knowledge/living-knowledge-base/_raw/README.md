# Raw sources — living-knowledge-base

Immutable captures (web pages, PDFs, pasted notes) that fed items in this topic. Once saved here, **never rewrite** — this is the provenance trail.

## Conventions

- Filename format: `<slug>-<YYYY-MM-DD>.<ext>` (e.g. `matuschak-evergreen-notes-2026-04-13.md`)
- For URLs: prefer `markitdown <url> -o _raw/<slug>-<date>.md` over copy-paste — deterministic, includes metadata
- For large files (PDFs): commit the markdown conversion, not the original binary (unless source is public + small)
- Cleaned versions (if needed) go alongside as `<slug>-cleaned.md`; never modify the `-raw.md` capture
